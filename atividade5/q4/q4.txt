4. Comente as seguintes frases à luz do ISP:

a. “Nenhum cliente deve ser forçado a depender de métodos que ele não
usa”;
    - Se uma interface contém métodos que não são relevantes para um cliente específico, a implementação dessa interface por parte do cliente cria uma dependência indesejada.

b. "Classes não devem ser forçadas a implementar interfaces que não usam":
    - Quando as interfaces são segregadas de maneira eficiente, uma classe pode escolher implementar apenas as interfaces que são relevantes para ela, evitando a necessidade de incluir métodos não utilizados.
